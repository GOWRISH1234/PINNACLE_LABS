How It Works

This tool uses XOR encryption to encrypt your images:

  1. Upload an image file
  2. Enter an encryption key
  3. Click "Encrypt Image" to encrypt the image
  4. Use the same key to decrypt the encrypted image
  5. Download the processed image when needed

Note: The encryption is done entirely in your browser. No images are uploaded to any server.
